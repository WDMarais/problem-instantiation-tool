f(x) = ax + b: Find x given f(x)

Page 1 of 2
Stage 1 asked: given $x$, find $f(x)$. This sheet asks the inverse: given $f(x)$, find $x$. Method: the rule gives $f(x) = ax+b$, so write $ax+b = c$ and solve. (Same framing governs every inverse question: $\arcsin$, $\log$, etc.)
A: Method: four steps
$\text{flib}(x) = -4x - 5$
$\text{given } \text{flib}(x) = -1\text{, find } x$
$-4x - 5 = -1$
$-4x = 4$
$x = -1$
$\text{sprock}(x) = -2x - 3$
$\text{given } \text{sprock}(x) = 5\text{, find } x$
$-2x - 3 = 5$
$-2x = 8$
$x = -4$
$\text{flib}(x) = 2x - 5$
$\text{given } \text{flib}(x) = -4\text{, find } x$
$2x - 5 = -4$
$2x = 1$
$x = \tfrac{1}{2}$
$\text{blorp}(x) = 2x - 2$
$\text{given } \text{blorp}(x) = 2k - 10\text{, find } x$
$2x - 2 = 2k - 10$
$2x = 2k - 8$
$x = k - 4$
$\text{quomp}(h) = 4h - 2$
$\text{given } \text{quomp}(h) = 10\text{, find } h$
$4h - 2 = 10$
$4h = 12$
$h = 3$
$\text{zing}(m) = -5m - 3$
$\text{given } \text{zing}(m) = -13\text{, find } m$
$-5m - 3 = -13$
$-5m = -10$
$m = 2$
