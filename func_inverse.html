Stage 1 asked: given $x$, find $f(x)$. This sheet asks the inverse: given $f(x)$, find $x$. Method: the rule gives $f(x) = ax+b$, so write $ax+b = c$ and solve. (Same framing governs every inverse question: $\arcsin$, $\log$, etc.)
A — Method: four steps
$\text{flib}(x) = -4x - 5,\quad \text{flib}(x) = -1$
$-4x - 5 = -1$
$-4x = 4$
$x = -1$
$\text{sprock}(x) = -2x - 3,\quad \text{sprock}(x) = 5$
$-2x - 3 = 5$
$-2x = 8$
$x = -4$
$\text{flib}(x) = -4x + 5,\quad \text{flib}(x) = -3$
$-4x + 5 = -3$
$-4x = -8$
$x = 2$
$\text{blorp}(x) = 2x - 2,\quad \text{blorp}(x) = -1$
$2x - 2 = -1$
$2x = 1$
$x = \tfrac{1}{2}$
$\text{sprock}(x) = 2x + 4,\quad \text{sprock}(x) = 5$
$2x + 4 = 5$
$2x = 1$
$x = \tfrac{1}{2}$
$\text{snorf}(x) = 3x + 3,\quad \text{snorf}(x) = 3m - 9$
$3x + 3 = 3m - 9$
$3x = 3m - 12$
$x = m - 4$
